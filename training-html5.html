Bref historique

  • 1991 HTML
  • 1994 HTML 2
  • 1996 CSS 1 + JavaScript
  • 1997 HTML 4
  • 1998 CSS 2
  • 2000 XHTML 1
  • 2002 Mise en forme sans tableau
  • 2004 WHATWG
  • 2005 AJAX
  • 2007 XHTML 2 boudé
  • 2007 Le W3C récupère les travaux du WHATWG
  • 2009 Abandon de XHTML 2
  • 2009 HTML 5 + CSS 3

Contexte
